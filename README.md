# 📚 GEL E-Book Store

A modern web-based E-Book Store developed using **HTML**, **CSS**, **JavaScript**, and **Firebase**. The application allows users to browse digital books and securely sign in using **Google Authentication** powered by Firebase Authentication.

---

# 📌 Overview

GEL E-Book Store is a responsive online bookstore prototype that demonstrates secure user authentication, cloud-based backend integration, and an intuitive user experience. The project showcases how Firebase services can be integrated into a web application for authentication and data management.

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/logeshwara554/GEL_E_bookStore.git
```

## Navigate to the project

```bash
cd GEL_E_bookStore
```

## Configure Firebase

1. Create a Firebase project.
2. Enable **Google Authentication**.
3. Copy your Firebase configuration.
4. Replace the Firebase configuration in the project with your own credentials.

---

# ▶️ Running the Project

Since this is a frontend web application, simply open the project using a local development server.

Examples:

```bash
python -m http.server
```

or use the **Live Server** extension in Visual Studio Code.

Open your browser and visit:

```text
http://localhost:8000
```
